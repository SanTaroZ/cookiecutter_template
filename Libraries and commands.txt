
open jupyter notebook:	jupyter notebook

para pasar las librerias a un archivo:	conda env export --from-history --file environment.yml

para crear desde un archivo yml las librerias:  conda env create --file environment.yml

para crear desde un archivo txt: conda install --f requirements.txt

cookiecutter 1.7.2 must be installed with the following: conda create --name name cookiecutter=1.7.2
